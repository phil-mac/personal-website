About this project

Lambda Build Week

This project was built as part of a Lambda School Build Week. Four students and one Project Lead collaborated together using Github, Slack and Trello to complete this project over the course of one week.

React Web App

Click the Sign Up button above to be taken to the React web app. This app allows for onboarding of a variety of user types, and displays a unique homepage to each type of user (admins, volunteers and students).

Other Features
